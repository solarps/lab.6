Задание: представленные задания оформить в виде бинарного файла.

Базовый уровень
Создать бинарный файл с информацией о предложенных к реализации телевизоры:
- название фирмы;
- размер по диагонали;
- стоимость;
Определить количество телевизоров фирмы «Samsung», размером более32 дюйма и распечатать сведения о них.

Средний уровень
Добавить в Memo или в StringGrid некоторое количество символов русского алфавита и записать их в бинарный файл. Переписать бинарный файл так, чтобы все символы были записаны заглавными буквами. Распечатать символы до и после изменения данных в файле.

Высокий уровень
Создать бинарный файл, компонентами которого является структура, содержащая следующие поля:
- фамилия и инициалы покупателя;
- дата покупки;
- общая стоимость приобретенного товара за первое полугодие;
- общая стоимость приобретенного товара за второе полугодие;
- начальный процент скидки на последующие приобретаемые товары;
Переписать бинарный файл так, чтобы процентная скидка была бы увеличена на 7%, если покупатель заплатил за предыдущий товар в первом и во втором полугодиях не менее 10000 грн.
